LORE SERVICES

OPAC

An electronic database containing details about different resources found in a library or collection of libraries, including books, authors, genres, journals, and other materials.

MEMBERSHIP

Library membership is granted to the University's employees, students, alumni and retirees and the public.The University's Library is automatically open to all enrolled students.

ILL

Interlibrary loan is a service that allows users of one library to obtain electronic documents and check out physical materials from library.We offer ILL and Document Delivery services to facilitate library users as well as member libraries.

SELF-CHECK MACHINE

This service is a great way to allows you to checkout and renew booksat the same time. It has a large monitor that easily and quickly takes you through the checkout process and users must keep the receipt for further reference.

RESERVATION

The reservations service allows you to request items on loan to others, or have a book delivered to another campus.

BOOK DROP MACHINE

Borrowed items may be returned in person during library hours, or drop them in the outside book drop located in front of the library main entrance.You can drop your books and it is collected three times a day.

LIBRARY ROOM BOOKING SERVICES

When renting a room from the library, users can make reservations. The staff can obtain precise information about the reservation made, and the user can use the services to determine the available time.

BOOK PURCHASE SUGGESTION FORM

In order for the library to purchase the book, users must fill out the given form. It can also satisfy the user's needs and broaden the library's collection.

ONLINE DATABASES

Online databases provided by libraries to their users so that users can find information quickly and accurately. It is also very useful because the user can use it at any time.
